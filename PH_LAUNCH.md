# Product Hunt Launch Kit

## Tagline
Give your website an AI-readable endpoint. LLMs and AI agents understand your content instantly.

## Description
Unweb makes any website AI-native. Publish structured content that AI agents can read — fast, clean, and reliable.

**Why?** The web was built for human eyes. But 80% of future traffic will be AI agents. They shouldn't have to parse your <div> soup.

**How it works:**
1. Register — create a free account
2. Publish your site content — one API call
3. Done — your site is AI-readable at unweb.app/llms/your-site

**Features:**
- 🔍 Universal URL extraction — any webpage to structured JSON
- 🤖 LLM-powered analysis — DeepSeek classifies and summarizes
- 📤 One-click publishing — permanent AI-readable endpoint
- 🔑 API key management — register, login, create keys
- 🌐 Open source — AGPL-3.0, self-hostable

**Tech:** Python FastAPI + Next.js + Supabase + DeepSeek
